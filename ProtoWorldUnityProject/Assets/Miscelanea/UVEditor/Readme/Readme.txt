A simple tool to add some UV's to object, or change the UV that are on them.

Installation
Just import the package into you project and you are good to go!
Tool is located under Window/UV Editor

Frequently Asked Questions (FAQ)
1) The UV Editor is located under \"Window\" 
2) On save or closing window a new asset is create wich contains the mesh and the new UV data
3) All is lost when you reimport your model (or press the revert button)
