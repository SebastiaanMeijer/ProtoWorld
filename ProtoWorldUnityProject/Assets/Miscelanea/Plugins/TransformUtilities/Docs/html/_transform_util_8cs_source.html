00008 \section About
00009 
00010 TransformUtil is a set of utilities for the Unity editor that
00011 allows you to easily nudge, rotate, flip and snap objects to a grid using
00012 keyboard shortcuts, or editor buttons.
00013 
00014 \section Features
00015  
00016  - Show a grid for any two axis XZ, YZ and YX independently or simultaneously
00017    with independent spacing in any direction and a separate color for
00018    each grid if wanted.
00019  - Nudge, Rotate, Flip and Snap using a custom editor window, menu items or
00020    keyboard shortcuts. Shortcuts can be customized by editing the 
00021    TransformUtilMenuItem.cs file (see
00022    http://unity3d.com/support/documentation/ScriptReference/MenuItem.html
00023    for information about how to set your own shortcuts)
00024  
00025 \section Usage
00026 
00027 In the menu, open the TransformUtil window under Edit --> TransformUtil -->
00028 Open Window.
00029 
00030 \image html window.png
00031 
00032 The Preferences tab will allow you to change the behaviour of the utilities,
00033 and the Controls tab contains buttons to nudge, rotate, flip and snap.
00034 
00035 As any other Unity window, this can be docked anywhere in your editor.
00036 
00037 All controls can also be activated from the menu, where you can also see the
00038 keyboard shortcut assigned to the function.
00039 
00040 \image html menu.png
00041 
00042 Note that the grid and snap functions are independent of Unitys internal grid
00043 and snap, and that control+mouse still uses Unitys internal grid and not
00044 the one provided by TransformUtil (you can however in most cases set the
00045 Unity grid to be of equal size).
00046 
00047 The default keyboard shortcuts are as follows:
00048         
00049  - Nudge Z: i
00050  - Nudge -Z: k
00051  - Nudge X: l
00052  - Nudge -X: j
00053  - Nudge Y: shift+i
00054  - Nudge -Y: shift+k
00055  - Snap XYZ: ,
00056  - Yaw Right: o
00057  - Yaw Left: u
00058  - Pitch Up: alt+o
00059  - Pitch Down: alt+u
00060  - Roll Right: shift+o
00061  - Roll Left: shift+u
00062  - Reset Rotation: alt+p
00063  - Flip X: alt+l
00064  - Flip Y: alt+i
00065  - Flip Z: alt+k
00066
